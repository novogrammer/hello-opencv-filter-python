# OpenCVの画像フィルターを試す（Python版）


## 前提

python 3.11
poetry

## 環境構築

```bash
poetry install
```

## 実行

cannyフィルターの場合
```bash
poetry run python ./hello_opencv_filter_python/filter_canny.py
```

参考： https://qiita.com/mo256man/items/8a9bb7bb66e107fd7153
mangaフィルターの場合

```bash
poetry run python ./hello_opencv_filter_python/filter_manga.py
```
